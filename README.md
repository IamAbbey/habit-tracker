## Getting Started

1. Clone this repository to your local machine:

   ```bash
   git clone https://github.com/IamAbbey/habit-tracker.git
   ```

2. Move to the cloned directory

   ```bash
   cd habit-tracker
   ```

3. Install dependencies:

   ```bash
   npm install
   ```

4. Run the development server:

   ```bash
   npm run dev
   ```
